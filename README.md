# Credit_Card_Financial_Report
Power BI Dashboard
Project Overview

This project is an interactive Power BI dashboard developed using credit card customer and transaction datasets. The dashboard provides insights into revenue, transaction volume, customer demographics, card category performance, and weekly business trends.

The project was completed as a hands-on learning exercise to strengthen Power BI, DAX, and data visualization skills.

Tools Used
Microsoft Power BI
DAX
Excel / CSV
Project Workflow
Data Preparation
Imported customer and transaction datasets from CSV files.
Cleaned and transformed data using Power Query.
Prepared datasets for reporting and analysis.
Data Modeling
Created relationships between customer and transaction tables.
Built a star schema model for efficient reporting.
DAX Calculations

Created measures and calculated columns for:

Revenue

Age Group Segmentation
Income Group Segmentation
